## git checkout

---

_Данная команда осуществляет переход из ветки в ветку, например, в ветку `master`_

```
git checkout master
```

_или в ветку `main-menu`_

```
git checkout main-menu
```

_Так же, можно создать ветку и сразу перейти в нее_

```
git checkout -b main-nav
```
