## git add

---

1. **Добавляем файлы в индекс.**

   _Данная команда подготавливает файлы для дальнейшей отправки_

   1.1. **Вы можете отправлять файлы в индекс по одному**

   ```
   git add readme.md
   git add index.html
   ... (и т.д.) ...
   ```

   1.2. **Или все сразу**

   _(в конце стоит точка)_

   ```
    git add .
   ```

   _или так_

   ```
   git add -A
   ```
